# simple-vue-input-kit

## Install
```
npm install simple-vue-input-kit
```

## Use
```javascript
import SimpleVueInputKit from 'simple-vue-input-kit';
Vue.use(SimpleVueInputKit);
```

### Available inputs
```
simple-input
simple-date
simple-select
simple-radio
simple-list-autocomplete
simple-ajax-autocomplete
```
